HEXO + Github 搭建博客

​ 想写博客很久了，因为自己的许多学习工作总是随着时间慢慢被遗忘，所以今天我开始了自己的第一篇博客。学生党，还请各位看客们多多批评指正！

​ 今天要介绍的就是怎么用Hexo和Github搭建博客，花了一下午才建起来的，记录分享一下。

步骤一：安装Note.js

​ 官网：https://nodejs.org/en/ 下载与自己电脑配置一致的版本。如图所示：

然后只要一路next安装就完成啦！为了验证已经装好了，同时按下win+R打开运行窗口，输入

1 node -v

2 npm -v

结果应如下图所示：

步骤二：Git的安装与配置

​ 下载地址：https://git-scm.com/downloads

​ 注意：git和Note.js都下最新版本比较好，如果版本差距较大的话后面会出错。刚开始就是因为这个才折腾了挺久。

​ 安装Git，如图：

然后一直next即可。安装完成后，在CMD窗口输入：

git –version

输出结果应为git的版本信息。

步骤三：github的配置

​ 前提：注册一个属于你的github账户，一定要注意username写自己的名字哟，不懂事的我给自己起了个淘宝名，后面博客的链接里就木有自己的名字了，伐开心~ 不在乎的小伙伴可以随便取的。

1.创建自己的代码库

​ 登录之后，点击右下角的New repository按钮

项目名必须遵守格式：账户名.github.io。如下图：用户名为xiyan77，则项目名必须为xiyan77.github.io，否则会出现问题哟，不要问我怎么知道的。另外，需要勾选Initialize this repository with a README

​ 然后在建好的项目中选中最右侧的settings，往下拉到GitHub Pages，结果如下图所示：

步骤四：安装Hexo

在合适位置创建一个新文件夹，我在E盘创建了一个blog文件夹，然后进入到该文件夹下：

然后依次输入以下命令：

npm install hexo -g ，安装Hexo

hexo -v ，检查安装是否成功，应该显示hexod的版本信息

hexo init ， 初始化该文件夹，最后应该出现 “Start blogging with Hexo！”

npm install , 安装所需要的组件

hexo g , 开始启动hexo

hexo s， 开启服务器，出现如下图结果：

复制该网页，在浏览器中打开，问题：假如页面一直无法跳转，那么可能端口被占用了。此时我们ctrl+c停止服务器，接着输入“hexo server -p 端口号”来改变端口号。

再次复制新的网页，就会出现如下图所示的炫酷画面啦！

步骤五：将Hexo与github page 联系起来

1.配置git的个人信息

​ 首先，在桌面右击，选择Git Bash Here，出现如下窗口：

输入如下命令：

git config –global user.name “xiyan77”

git config –global user.email “888888888@qq.com“

ssh-keygen -t rsa -C “888888888@qq.com“

注意：xiyan77 应改为你的用户名，邮箱也改为你的邮箱。第三行是设置密码，如果不想设置密码，输入三次enter即可。

eval “$(ssh-agent -s)”， 添加密钥到ssh-agent

ssh-add ~/.ssh/id_rsa， 添加生成的SSH key到ssh-agent

2.登录github，添加ssh

点击下图中的settings:

添加ssh，点击New SSH key:

新建一个new ssh key，将id_rsa.pub文件里的内容复制上去（目录：C:\Users\刘梦雅.ssh，默认安装的话就应该在类似的目录中），保存后如上图。输入ssh -T git@github.com，测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了。

3.配置Deployment，在最开始新建的文件夹中，找到_config.yml文件，修改repo值（在末尾）

步骤六：新建博客

新建一篇博客，在cmd执行命令：hexo new post “博客名”，在文件夹_posts目录（E:\blog\source_posts）下将会看到已经创建的文件。

在生成以及部署文章之前，需要安装一个扩展：npm install hexo-deployer-git –save

使用编辑器（我用的是Typora）编好文章，那么就可以使用命令：hexo d -g，生成以及部署了

部署成功后访问你的地址：http://用户名.github.io。那么将看到生成的文章。
